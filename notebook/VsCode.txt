vs code相关插件、快捷键：

【页面格式化】
shift + alt + f

【选择多行内容】
shift + alt + 鼠标左键 + 上下移动
shift + alt + 鼠标左键 + 上下移动 + End （移动到所选行的行尾）

【移动到行首/行尾】
行首：Home
行尾：End

【保存文件（单个）】
ctrl + s

【保存多个更改的文件】
ctrl + k （等待1秒）  + s

【打开vs code 键盘快捷方式】
ctrl + k + s （连续按 不需要等待）

【快速复制当前行的内容】
shift + alt + ↓

【回退】
ctrl + z

【文件内查找】
ctrl + f

【直接跳转到下一行（硬性回车）】
ctrl + enter
